# 页面与交互清单

## 1. Dashboard

首页展示求职概览。

核心模块：

- 总投递数
- 本周投递数
- 回复数
- 面试数
- Offer 数
- 当前状态分布
- 最近需要跟进的岗位
- 最近面试安排

第一版可以用简单统计卡片 + 列表。

## 2. Applications

岗位列表页。

功能：

- 新增岗位
- 搜索公司 / 岗位
- 按状态筛选
- 按渠道筛选
- 按投递日期排序
- 进入岗位详情

列表字段：

- 公司
- 岗位
- 渠道
- 状态
- 投递日期
- 下次跟进
- 关联简历版本

## 3. Application Detail

岗位详情页。

模块：

- 基本信息
- JD 原文
- JD 分析
- 关联简历版本
- 简历匹配建议
- 面试记录
- 备注

主要操作：

- 编辑岗位
- 更新状态
- 分析 JD
- 生成匹配建议
- 新增面试记录

## 4. Resume Versions

简历版本管理页。

功能：

- 新增简历版本
- 编辑简历内容
- 标记适用方向
- 查看关联岗位

字段：

- 名称
- 目标岗位
- 核心卖点
- 正文 / 文件路径

## 5. Interviews

面试记录页。

功能：

- 按时间查看面试
- 按标签查看问题
- 查看高频问题
- 查看薄弱点

MVP 可以先只做按时间列表。

## 6. Analytics

统计页。

第一版指标：

- 投递总量
- 回复率
- 面试率
- Offer 数
- 渠道分布
- 状态分布
- 每周投递趋势

第二版再做：

- 技术关键词频率
- 简历版本效果
- 岗位方向效果

## 7. Settings

设置页。

第一版：

- AI Provider
- API Key
- 本地模型地址
- 数据导入 / 导出

AI Provider：

- None
- OpenAI Compatible
- Ollama
