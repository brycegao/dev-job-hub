# 程序员求职作战台

> Local-first job hunt dashboard for Chinese developers. Paste JD, track applications, resumes, interviews, and get actionable insights — all in one private workspace.

粘贴 JD，自动建档。追踪投递漏斗、简历版本、面试复盘，AI 辅助生成面试准备包。纯本地运行，数据不离开浏览器。

## Why

找工作时，信息散落在招聘软件、微信、简历文档和 AI 对话里：

- JD 复制过一次，过几天就找不到原文
- 投了很多岗位，不知道哪些渠道有效、问题出在哪
- 简历版本不断修改，但很难关联到真实面试反馈
- 面试问题没有系统复盘，薄弱点反复出现
- AI 能帮忙，但缺少自己的 JD、简历和历史面试上下文

这个工具把这些信息收束到一个本地工作台，让求职过程变得**可记录、可分析、可行动**。

## Highlights

### 输入减负
- **JD 粘贴自动建档** — 粘贴招聘页面文本，自动提取公司、岗位、渠道、薪资、城市、工作方式，填充高亮提示
- **渠道自动补全** — 记住历史渠道，输入时自动推荐
- **状态一键切换** — 列表中点击状态标签直接流转，无需打开详情
- **面试标签智能填充** — 新建面试时自动从 JD 关键词生成问题标签

### 输出智能
- **今日行动台** — 聚合到期跟进、超期无反馈、即将面试、待复盘和资料补全任务，按优先级排序
- **瓶颈洞察** — 自动识别低回复率、低面试转化率、渠道过于集中等问题，给出具体改进建议
- **跨面试薄弱点分析** — 聚合所有面试的薄弱点，找出高频重复项，如"你在 3 场面试中「Flutter 性能优化」表现薄弱"
- **智能提醒** — 停滞岗位主动提醒跟进，即将面试时提醒历史薄弱点与 JD 重合
- **简历匹配建议** — 基于 JD × 简历交叉分析，按关键词类别（技术/能力/领域/加分项/风险）给出差异化建议

### 核心功能
- **JD 关键词分析** — 本地规则识别技术栈、业务方向、能力要求、加分项和风险点
- **面试准备包** — 无需 API Key 也能生成可复制到 ChatGPT / DeepSeek / 豆包的 Prompt
- **可选 AI Provider** — 支持 OpenAI-compatible API / Ollama，页面内直接生成建议
- **渠道转化漏斗** — 按渠道可视化投递→回复→面试→Offer 转化率
- **日历导出** — 面试时间一键导出 .ics 文件到系统日历
- **本地优先** — 无登录、无后端、无云同步，数据保存在当前浏览器 IndexedDB

## Who Is It For

适合正在求职、需要同时管理多个岗位的开发者：

- Android / Flutter / iOS / React Native 移动端工程师
- 前端 / 后端 / 全栈 / AI 应用开发者
- 通过 BOSS 直聘、脉脉、猎聘、拉勾、内推等渠道投递的人
- 想复盘投递效率、面试问题、简历版本效果的人

不适合企业招聘团队、多人协作 ATS、批量投递或招聘网站爬虫场景。

## Product Flow

```
粘贴 JD → 自动建档 → 关联简历 → 分析匹配 → 准备面试 → 复盘薄弱 → 调整策略
```

1. 进入"岗位"，点击新增，粘贴招聘 JD — 自动提取公司、薪资、城市等字段
2. 关联投递用的简历版本，生成简历匹配建议
3. 查看"概览"今日行动台，处理跟进、面试准备任务
4. 记录面试邀约、地点、下一轮安排和结构化复盘
5. 查看"统计"瓶颈提示和高频薄弱点，调整求职策略
6. 回到岗位列表，点击状态标签快速更新投递进度

## Privacy

默认不需要登录，也不需要服务器。

- 岗位、JD、简历内容、面试复盘保存在浏览器 IndexedDB
- API Key 只保存在当前浏览器 `localStorage`
- API Key 不会进入 JSON 导入导出数据包
- 不内置云同步，不上传用户数据
- 使用外部 AI Provider 时，请自行确认对应服务的数据策略

## Current Features

### 岗位管理
- 新增、编辑、删除岗位，支持状态筛选和全文搜索
- JD 粘贴自动提取公司、岗位、渠道、薪资、城市、工作方式
- 状态一键切换（列表中点击状态标签直接流转）
- 岗位关联简历版本

### 简历管理
- 多简历版本管理，每个版本填写核心卖点和简历正文
- 简历与 JD 交叉匹配，按类别生成差异化建议

### 面试管理
- 面试邀约记录（时间、地点/会议链接、下一轮安排）
- 结构化复盘（问题、标签、薄弱点、亮点、行动项、评分）
- HR 邀约确认自动更新岗位状态
- 日历导出（单条 / 批量）
- AI 面试回答包

### 分析与洞察
- 今日行动台（优先级排序的待办事项）
- 智能提醒（停滞岗位、面试前薄弱点重合）
- 投递漏斗可视化（总览 + 按渠道）
- 瓶颈识别（低回复率、低面试率、渠道集中）
- 跨面试高频薄弱点分析
- 状态分布和渠道分布统计

### 数据管理
- JSON 导入导出（全量备份恢复）
- 一键加载示例数据
- 可选 AI Provider 配置（OpenAI compatible / Ollama）

## Tech Stack
